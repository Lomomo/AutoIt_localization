AutoIt v3 首頁

緒論

AutoIt v3 是用以編寫並生成具有 BASIC 語言類型腳本的免費軟體程式，它被設計用來在Windows GUI (使用者界面) 中進行自動化。利用它可以模擬快速鍵、移動滑鼠和視窗/控制項的互動，而這是其它語言所無法做到或尚無可靠方法實現的 (如 VBScript 和 SendKeys)。AutoIt 亦具備小巧、高度自我啟發性，及可以不受框架限制於所有 Windows 版本上執行，而且絕不會有煩人的 "runtimes" 的編寫需要！

AutoIt 最初是為 PC 的 "批次處理" 而設計，用於對數千台 PC 進行同樣的設定。隨著時間的推移，它已成為一種強大的語言，支援複雜的陳述式、使用者自定函式、迴圈和資深程序員期望的所有內容。主要特色:

  • 類似- BASIC 語言且很容易學習
  • 可模擬按鍵敲擊及滑鼠動作
  • 可操作視窗及處理程式
  • 可與所有標準視窗控制項互動
  • 腳本可被編譯成獨立的單一執行檔
  • 建立 Graphical User Interfaces (GUIs) 視窗介面
  • COM 的支援
  • 規則陳述式
  • 直接呼叫外部 DLL 及 Windows API 函式庫
  • 可腳本化 RunAs 函數
  • 鉅細彌遺的說明書及龐大社群論壇基礎支援
  • 相容於 Windows XP SP3 / 2003 SP2 / Vista / 2008 / Windows 7 / 2008 R2 / Windows 8 / 2012 R2 等系統
  • 支援 Unicode 及 x64
  • 讓人放心的數位化簽署
  • 適用於Windows用戶帳戶控制（UAC）

AutoIt 被設計得盡可能短小精悍且不用依賴外部 DLL 檔或新增登錄碼即可獨立執行。使用 Aut2Exe 這個工具即可把腳本檔案編譯為可獨立執行的檔案。

我們還更新了 AutoIt 呼叫 AutoItX 的已模組化 COM 及 DLL 版本，AutoItX 將使得您可以加入一些 AutoIt 獨有的特性到您最常用的腳本語言或程式設計語言中去！

最重要的是，AutoIt仍然是免費的，但是如果你想要支持AutoIt，不論是時間或著金錢，你可以至AutoIt的首頁進行捐助。

詳細特色

Basic 語言類型及豐富的函式庫設定

AutoIt 具備 BASIC 語言類型，此意味大部份的人只要曾經撰寫過腳本或使用高階語言，將會很容易對它上手。

雖然它最初目地僅在於簡單的自動化工具，如今 AutoIt 擁有函式庫及各種功能，使的它被當成通用的腳本語言被使用。 (如您所料就跟神奇的自動操作一樣！)。語言功能內含：

  • 常見的高階函數功能類型，如迴圈、運算式分析。
  • 豐富眾多的字串處理功能以及相容於 Perl 規則陳述式編譯引擎 (使用了 PCRE 函式庫)。
  • COM 的支援
  • 呼叫 Win32 及第三方 DLL APIs。

 

具備語法高亮功能的組建編輯器

AutoIt 帶來了可自訂 SciTe 的 "LITE" 版，屆時編輯將更為輕鬆。使用者也可以 下載完整的 SciTe 版本 它還附加了外掛套件工具以簡化編輯工作。

 

獨立小巧且精悍

AutoIt 是一種小巧且不依靠於大量Runtimes 語言如 .NET 或 VB 的單獨程式。你所需要執行的 AutoIt 腳本套件就是 AutoIt 執行檔 (AutoIt3.exe) 和腳本。編寫好的腳本也可以腳本組建編譯器 Aut2Exe 編譯成獨立的可執行檔案。

 

國際化 64-位元 版本的支援

AutoIt 是完完全全的 Unicode 且亦內含了 x64 版本所有的主要元件！至於其它數之不清的眾多腳本語言你舉例的出嗎？

 

模擬鍵盤和滑鼠動作

我們花了許多時間用在對鍵擊和滑鼠動作的模擬功能的最佳化上，使其在目前所有的作業系統上都可以做到盡可能的精確。在這一版本中滑鼠動作的功能也表現得更像 「人類」了，甚至可以用來建立靈活的示範腳本。所有模擬滑鼠和鍵盤動作的程式在模擬動作的 "速度" 及其它功能上都是高度可自訂的。

 

視窗動作管理

您可以對視窗進行搬移、隱藏、顯示、調整大小、啟動、關閉以及其它所有您想要實現的動作。視窗還可以利用標題、視窗文字、大小、位置、類型，甚至系統內建的 Win32 API 控制碼來互相關聯。

 

控制項

直接取得編輯框、複選框、清單框、選擇框、按鈕、狀態列上的訊息，並且作用於這些控制項而不必擔心按鍵動作落空。 甚至還可以作用於非活動視窗上的控制項！
