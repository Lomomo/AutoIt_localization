函數說明（Function Reference）

關閉指定視窗。

關閉指定視窗。

WinClose ( "title" [, "text"] )

參數

title 要關閉的視窗標題，請檢視：視窗標題特別定義 (Title special definition)。
text [選擇性參數] 要關閉的視窗文字。請檢視：Text special definition

返回值

成功： 返回 1 。
失敗： 如視窗未找到返回 0。

附註

此函數會傳送一個關閉訊息給視窗，其結果會依視窗而有不同 (它可能會詢問您是否儲存資料...等)。要強制關閉視窗，請使用 WinKill 函數，如有數個符合條件的視窗，則以最近開啟的視窗將其關閉。

相關

ProcessClose, WinActivate, WinExists, WinKill, WinMove, WinSetState, WinTitleMatchMode (Option)

範例

#include <MsgBoxConstants.au3>

Example()

Func Example()
    ; Run Notepad
    Run("notepad.exe")

    ; Wait for 2 seconds to display the Notepad window.
    Sleep(2000)

    ; Close the Notepad window using the classname of Notepad.
    If WinClose("[CLASS:Notepad]", "") Then
        MsgBox($MB_SYSTEMMODAL, "", "Window closed")
    Else
        MsgBox($MB_SYSTEMMODAL + $MB_ICONERROR, "Error", "Window not Found")
    EndIf
EndFunc   ;==>Example
